private arch linux dotfiles

```sh
git clone git@github.com:Mark-Marks/arch-dotfiles.git
```
```sh
cd arch-dotfiles
```
```sh
chmod +x ./setup.sh
```
```sh
./setup.sh
```

script usage:
```
-h = view script usage
-p = don't install packages
-y = don't install yay
-n = don't install nerd fonts complete jetbrains
-o = don't install oh-my-posh
-s = don't do symlinks
```
